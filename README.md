# OCR Text Extractor

## Description
This project uses Optical Character Recognition (OCR) to scan digital images of book pages and extract the text into raw .txt files.

## Technologies Used
- Python
- Tesseract OCR 
- OpenCV 

## Features
- Image text detection
- Text extraction
- Output to .txt file
- Works with scanned book images

## How It Works
The program processes image files, detects text using OCR, and saves the extracted content into a text file.
